Coffe is a project about coffee and bar, where users have to shout if the bar is full or the coffee is full, featuring smooth animations and a clean design.

## Tech Stack

- [Nuxt 4](https://nuxt.com) - The Intuitive Web Framework
- [TailwindCSS](https://tailwindcss.com) - A utility-first CSS framework

## Getting Started

Required to install: [Node.js](https://nodejs.org/)

Install dependencies:

```bash
bun install
```

Run Development

```bash
bun dev
```

---

> [!NOTE]
> if not using `bun` just delete `bun.lockb`, and switch to another package manager like:

```bash
# npm install
# yarn install
# pnpm install
```
